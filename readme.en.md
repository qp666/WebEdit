# WebEdit

A Chrome extension that lets you **freely edit** content on any web page.

---

## 📥 Installation

### 🛒 Option A: Chrome Web Store (recommended)

1. Open the [Chrome Web Store](https://chromewebstore.google.com/).
2. Search for **WebEdit** and click **Add to Chrome** to install.

### 📦 Option B: Load unpacked from GitHub

1. On the project’s GitHub page, open **Releases**, download the latest archive, and extract it.
2. In Chrome, go to `chrome://extensions/` and turn on **Developer mode** (top right).
3. Click **Load unpacked** and select the folder that contains `manifest.json` (usually the **`WebEdit`** folder inside the release).
4. Optionally **pin** the extension to the toolbar for quick access.

> **🔄 Updating a sideloaded build**: Replace the extracted folder with the new release (or pick the new folder), then open `chrome://extensions/` and click **Reload** on this extension.

---

## ✨ Features

1. **🖱️ Toolbar icon** — Click the extension icon to turn edit mode on or off.
   - **🔵 Blue icon**: Edit mode is **on**.
   - **Gray icon**: Edit mode is **off**.
2. **📋 Context menu** — Right-click anywhere on the page and choose the toggle for edit mode.
3. **⌨️ Shortcut** — Press `Alt` + `Shift` + `D` to toggle edit mode quickly.
